# ASUS-ZENBOOK-UX410-DSDT-PATCH
A collection of DSDT/SSDT patches and tools for the ASUS UX401 laptop for macOS 10.12.6+

#### What you should expect works

* Audio
* Keyboard
* Trackpad
* Battery
* Fn-keys
* Brightness
* Sleep
* HDMI

#### Will never work

* Factory Intel WiFi card.
